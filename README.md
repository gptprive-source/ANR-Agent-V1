# ANR-Agent-V1

## Objectif
Créer l’Interphone V1 Visio basé sur Daily.co, en se servant du repo ANR comme référence **sans jamais le modifier**.

## Contexte
- ANR existant = référence seulement
- V1 = visio uniquement (pas d’ouverture de porte, pas de relais colis)
- Agent doit découper le projet en petites tâches simples
- Tester et debugger chaque tâche avant de passer à la suivante

## Workflow de l’agent
1. Lire le repo ANR pour récupérer les éléments utiles
2. Générer module backend / frontend pour Daily.co
3. Créer tests unitaires pour chaque module
4. Valider et corriger les bugs
5. Passer à la tâche suivante
